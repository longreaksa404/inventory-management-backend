# Inventory Management System (Backend)

A production-ready Django REST Framework backend for multi-warehouse inventory management with role-based access control.

## Features

### Security & Access
- JWT authentication (access/refresh tokens)
- Role-based permissions (Admin, Manager, Staff)
- Activity logging

### Core Functionality
- Multi-warehouse stock management
- Product catalog with unique SKU enforcement
- Supplier management
- Purchase & sales order workflows
- Stock transfers between locations
- Soft deletion for data integrity

### Reporting & Insights
- Real-time inventory valuation
- Low-stock alerts
- Transaction audit trails
- Category/supplier analytics
